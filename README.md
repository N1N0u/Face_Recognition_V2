# Face Recognition V2 🔥

Real-time face recognition using:
- DroidCam (mobile camera)
- MTCNN (face detection)
- FaceNet (face embeddings)

## Features
- Real-time detection
- Face recognition
- Mobile camera integration

## Tech Stack
- Python
- OpenCV
- MTCNN
- FaceNet

## Run

pip install -r requirements.txt

python src/main.py
